# 🎮 The Last Stronghold

A third-person action tower defense game built in Unreal Engine 5, where players must defend a powerful arcane obelisk against waves of enemies using combat abilities and strategic defenses.

---

## 📖 Overview

**The Last Stronghold** is a dark fantasy experience that blends fast-paced third-person combat with strategic tower defense mechanics. Players take on the role of a guardian tasked with protecting the last remaining source of magical power from relentless enemy forces.

The game emphasizes:
- Real-time combat
- Strategic resource management
- Wave-based survival gameplay

---

## 🚀 Features

- ⚔️ **Third-Person Combat System**
  - Magic-based projectile attacks
  - Combo attack system
  - Enemy hit reactions and damage handling

- 🧠 **AI Enemy System**
  - Dynamic enemy targeting (player vs. objective)
  - Multiple enemy types with unique behaviors
  - Boss encounters with advanced attack patterns

- 🏰 **Tower Defense Mechanics**
  - Deployable arcane turrets
  - Area-of-effect crystal defenses
  - Strategic placement between waves

- 🌊 **Wave System**
  - Progressive difficulty scaling
  - Multiple levels with unique environments
  - Boss waves integrated into progression

- 💎 **Resource System**
  - Enemies drop resources upon defeat
  - Spend resources to build defenses
  - Risk vs. reward gameplay loop

---

## 🎯 Gameplay Objective

- Defend the arcane obelisk at all costs  
- Survive all enemy waves  
- Strategically build defenses between rounds  
- Defeat bosses to progress  

---

## 🧠 Design Approach

The game was designed using a **modular Blueprint architecture** to ensure scalability and maintainability. Systems communicate through interfaces rather than direct references, reducing coupling and improving reusability across different game components.
